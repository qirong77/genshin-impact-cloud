## 本地启动项目 Project Setup

### Install

```bash
$ npm install
```

### Development

> node > 18

```bash
$ npm run dev
```

### 打包 Build

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
